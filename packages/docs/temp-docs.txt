src/: Contiene el código fuente del proyecto. Aquí se encuentra la lógica principal de la aplicación.

tests/: Almacena las pruebas unitarias e integración para asegurar que el código funcione correctamente.

docs/: Incluye documentación relevante, como guías de uso, instalación y desarrollo.

assets/: Guarda archivos estáticos como imágenes, íconos y fuentes.

bin/: Contiene scripts ejecutables o archivos binarios generados.

config/: Configuración de la aplicación, como archivos de entorno o configuración de base de datos.

venv/ o env/: Almacena el entorno virtual para proyectos en Python.

build/: Archivos generados, como compilaciones o distribuciones del proyecto.
